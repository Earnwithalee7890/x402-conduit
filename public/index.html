How x402 Works

HTTP 402 was reserved for "future use" since 1997. x402 finally gives it purpose ‚Äî native payments at the protocol level.

01
üîç

Discover

Agents call GET /api/v1/discover to browse the full API catalog with pricing, docs, and availability ‚Äî completely free.

02
üì°

Request

Send a standard HTTP request to any paid endpoint. The server responds with HTTP 402 and payment requirements in the header.

03
üíé

Pay

The x402-stacks interceptor automatically signs a STX transfer, submits it to the facilitator, and retries the request with proof of payment.

04
‚ö°

Receive

The server verifies payment and returns your data. The entire flow happens in a single HTTP round-trip ‚Äî completely transparent to the agent.

Integration

Ship in 5 Minutes

Add x402 payments to any API with one middleware line, or consume paid APIs with the axios interceptor.

agent.js x402-stacks 
// npm install x402-stacks axios
import axios from 'axios';
import { wrapAxiosWithPayment, privateKeyToAccount } from 'x402-stacks';

// Create account from private key
const account = privateKeyToAccount(process.env.PRIVATE_KEY, 'mainnet');

// Wrap axios ‚Äî all 402 payments are now automatic
const api = wrapAxiosWithPayment(
  axios.create({ baseURL: 'https://x402.market' }),
  account
);

// 1. Discover available APIs (free)
const catalog = await api.get('/api/v1/discover');

// 2. Use paid APIs ‚Äî payment handled automatically
const weather = await api.get('/api/v1/weather?location=Tokyo');
// ‚Üí Interceptor sees 402, signs STX transfer, retries ‚Üí 200 OK

const price = await api.get('/api/v1/price?symbol=BTC');
// ‚Üí 0.005 STX paid automatically

console.log(weather.data);  // ‚úÖ Full weather data
console.log(price.data);    // ‚úÖ Real-time BTC price
server.js Express.js 
// npm install x402-stacks express
import express from 'express';
import { paymentMiddleware, STXtoMicroSTX } from 'x402-stacks';

const app = express();

// Monetize any endpoint with one middleware line
app.get('/api/premium-data',
  paymentMiddleware({
    amount: STXtoMicroSTX(0.01),
    payTo: 'YOUR_STACKS_ADDRESS',
    network: 'mainnet',
    facilitatorUrl: 'https://x402-facilitator.x402stacks.xyz',
    description: 'Premium data access',
  }),
  (req, res) => {
    // Only reached after payment is verified
    res.json({ data: 'Your premium content' });
  }
);

// Without payment ‚Üí 402 Payment Required (automatic)
// With valid payment ‚Üí 200 OK + data
// No billing logic, no auth code, no API keys

app.listen(3402);
